⚙️ Features
High-performance API using FastAPI
Serverless deployment with AWS Lambda
AI integration for generating responses
Lightweight and auto-scalable architecture
🛠️ Tech Stack
FastAPI (Python)
AWS Lambda + API Gateway
OpenAI API (or any LLM)
Mangum (for ASGI to Lambda adaptation)
🚀 How it Works
Client sends request to API Gateway
AWS Lambda triggers FastAPI app
Backend processes input and calls AI model
Response is returned to the client
